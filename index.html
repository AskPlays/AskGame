AskGame
Installation

Either download the repository or link to the file hosted on github pages.
Add import GAME from 'https://askplays.github.io/AskGame/AskGame.js' to your js file
To make import work, make sure to use <script type="module"> in your html

Usage

Use GAME.init(init, loop) to setup your game

Example 

import GAME from 'https://askplays.github.io/AskGame/AskGame.js';

GAME.init(()=>{
    GAME.size(400, 400);
}, (time, delta)=>{
    console.log(time);
});
